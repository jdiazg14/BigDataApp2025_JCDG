# Proyecto de BIGDATA para la maestría en Analítica de Datos
Aplicación Normatividad del Ministerio de Agricultura y Desarrollo Rural

# Autor
Nombre:  Juan Carlos Díaz González
Email:   jdiazg14@ucentral.edu.co

# Descripción
Aplicativo web que descarga normatividad pública del portal del Ministerio de Agricultura a través de Web Sacraping.
Se realiza busqueda semántica en toda la documentación tratada previamente con procesamiento de lenguaje natural.
